# 📘 融合Qwen微调与RAG的材料知识智能问答系统

## 基于 LoRA 领域微调与混合检索增强生成（Hybrid RAG）

---

## 摘要

通用大语言模型在自然语言理解与生成任务中展现出强大能力，但在材料科学等高度结构化、物理约束严格的科学领域中，仍普遍存在事实性幻觉、数值错误与机理解释不严谨等问题。

本项目构建了一套**材料科学知识增强大模型系统**，通过将结构化材料数据库与大语言模型融合，提出：

* 面向材料物理语义的指令数据重构方法
* 基于 LoRA 的领域对齐微调策略
* 稠密向量 + 稀疏关键词的混合检索增强生成框架
* 面向科学问答的自动化多维评测体系

实验结果表明，该系统在材料结构参数、热力学稳定性与机理解释等任务上显著降低了幻觉率并提升了物理一致性。

---

## 1. 研究背景与动机

材料科学数据具有以下典型特征：

* 高度结构化（晶体结构、对称性、物性参数）
* 强物理约束（稳定性判据、能量关系）
* 高精度数值依赖

而通用大模型存在：

* 混淆晶体结构类型
* 错误生成形成能、Ehull 等关键指标
* 缺乏物理机制约束推理能力

因此，本研究提出构建**融合Qwen微调与RAG的材料知识智能问答系统**，实现：

> 结构化物理语义注入 + 证据增强生成 + 微调

---

## 2. 系统总体架构

```text
Materials Project 数据库
        ↓
全量数据抽取
        ↓
物理语义重构
        ↓
LoRA 指令微调
        ↓
混合向量知识库
        ↓
检索增强生成（RAG）
        ↓
自动化科学评测
```

---

## 3. 数据构建方法

### 3.1 材料数据库采集

* 基于 Materials Project 获取所有材料摘要数据
* 包含：

  * 晶体结构信息
  * 空间群与对称性
  * 带隙
  * 形成能
  * 能量高于凸包（Ehull）
  * 稳定性标签

最终构建高覆盖度材料科学基础语料库。

---

### 3.2 物理语义重构

将原始结构化字段转化为：

* 可检索的科学描述文本
* 显式物理属性头信息
* 保留数值与单位约束

示例结构：

```
FORMULA | SpaceGroup | BandGap | FormationEnergy | Ehull | Stability | Lattice Parameters
```

该设计显著提升材料级精确命中能力。

---

## 4. 领域对齐 LoRA 微调

采用两部分数据进行高效微调：

### 部分一：知识注入

* 长文本材料科学知识
* 物理概念解释
* 数据驱动事实对齐

### 部分二：任务强化

* 具体材料参数问答
* 结构对比推理
* 稳定性判据分析

实现模型从“语言模型”向“材料科学专家模型”的迁移。

---

## 5. 混合检索增强生成（Hybrid RAG）

### 5.1 稠密语义检索

* 基于 BGE-M3 向量嵌入
* 捕捉材料语义相似性

### 5.2 稀疏物理关键词检索

* 基于 SPLADE 稀疏编码
* 强化公式、数值与专业术语匹配

### 5.3 精排机制

* Cross-Encoder 联合建模
* 消除“语义接近但物理错误”召回项

---

## 6. 自动化评测体系

对比模型原始通用大模型，领域微调模型，微调 + 检索增强模型三种模型在不同维度上的优劣
评测维度：

* 事实正确性
* 物理机制解释深度
* 完整性
* 表达清晰度

## 7. 实验结论（摘要）

| 模型       | 幻觉率 | 事实准确性 | 机理解释 |
| -------- | --- | ----- | ---- |
| Base     | 中   | 低     | 低    |
| LoRA     | 中   | 中     | 中    |
| LoRA+RAG | 低  | 高     | 高    |

混合检索增强显著提升材料科学问答可靠性。

## 10. 学术价值

本研究验证了：

* 大模型可通过结构化科学数据进行有效领域对齐
* RAG 在高精度科学场景中的必要性
* 混合检索可显著降低事实性幻觉

本研究为大语言模型在材料科学等高结构化专业领域中的工程应用提供了一种可行的系统化实现路径。未来工作可进一步扩展训练数据规模与材料属性覆盖范围，并探索多模态材料数据融合分析方法，以构建更加智能与可靠的材料研发辅助系统。
